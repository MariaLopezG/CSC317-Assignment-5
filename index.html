How to Apply CSS to HTML

To include an external CSS file in an HTML file, you need to use the <link> element within the <head> section of your HTML document. Here is an example: 


        <head>
            <link rel="stylesheet" href="styles.css">
        </head>

How to Use JavaScript in a Webpage

To link to an external JavaScript file, you use the <script> element with the src attribute. Here is an example: 


        <script src="script.js"></script>
To write JavaScript code directly within an HTML file, you can include it inside a <script> element. Here is an example: 

        <script>
            console.log('Hello, world!');
        </script>
